#  She Can Foundation Website

A modern and responsive NGO website created for **She Can Foundation**, an organization dedicated to empowering women and young girls through education, leadership, mentorship, and community support.

---

##  Features

- Responsive modern NGO website
- Beautiful orange-red theme
- Hero section with empowering message
- About Us section
- Programs & initiatives section
- Organization gallery
- Join Us section
- Donation section
- Contact section with social icons
- Smooth animations and hover effects
- Mobile-friendly design

---

##  Technologies Used

- HTML
- CSS
- Google Fonts
- Font Awesome
